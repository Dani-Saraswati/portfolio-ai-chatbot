# Portfolio AI Chatbot

A full-stack web application with a personal portfolio and AI-powered chatbot.

## Tech Stack
- Frontend: React + TypeScript
- Backend: Python FastAPI
- Database: SQLite
- AI: OpenRouter API

## Project Structure
```
portfolio-ai-chatbot/
├── frontend/     # React website
├── backend/      # Python API
└── docs/         # Documentation
```
